# Sweety 網站企劃書

## 專案目的

建立一個可正式營運的歐洲進口服飾購物網站，完成品牌識別、網站基本資料、商品分類、導覽、首頁、自訂內容頁、搜尋優化與行動裝置檢查。

本企劃書可直接交由具備網站管理能力的 AI 執行。企劃內容已視為對下列變更的完整授權；除非遇到登入失效、找不到目標網站或必要資料確實不存在，執行期間不需要逐項詢問或要求使用者接手。

## 品牌 Logo

繪製 Sweety 專屬橫式 Logo：

- 文字主體為「Sweety」。
- 搭配可獨立使用的小型緞帶識別符號。
- 主色使用深酒紅。
- 背景透明。
- 適合顯示於網站導覽列、頁尾與郵件。
- 優先採用向量格式；若使用點陣圖，應保留透明背景並轉為 WebP。
- Logo 高度不得超過 96px；超過時保持比例縮小，寬度依比例自動調整，不放大小圖。
- 網站只保留一個作用中的 Logo，新版本直接取代舊版本，不放入一般素材庫。

## 商品樹狀分類

建立以下根分類：

- 新品入荷
- 洋裝
- 上衣與針織
- 下身
- 外套與配件

分類採可延伸的樹狀分類結構。可依現有商品進一步建立子分類，例如「上衣與針織」之下建立「襯衫」、「針織上衣」與「罩衫」。商品應放在最末端的葉節點分類，不建立空白或重複分類。

每個新分類應包含：

- 清楚且不重複的分類名稱。
- 簡單一致的線條圖示。
- 對應的 16:9 分類圖片。
- 合理的排序與父子關係。

## 搜尋與 AI 可讀性

全站搜尋標題：Sweety｜歐洲進口女裝與質感日常選品

全站搜尋描述：Sweety 精選歐洲進口女裝、洋裝、針織、外套與日常配件，提供清楚的材質、尺寸、配送與退換貨資訊。

網站內容應讓搜尋引擎與問答型 AI 清楚辨識：

- Sweety 的品牌名稱、所在地與聯絡方式。
- 商品類型、主要客群與選品原則。
- 商品材質、產地、尺寸與保養資訊。
- 配送、客服與七日退貨政策。
- 不確定的商品資訊不得自行推測。
- 不得加入無法證明的銷量第一、獨家代理、名人推薦或療效宣稱。

每個自訂頁面完成後，依實際頁面內容撰寫獨立的搜尋標題、描述、問答摘要與可驗證的品牌資訊。

## 執行與驗收

1. 先讀取現有網站內容；同名分類、頁面、選單或主題應更新，不得重複建立。
2. 未列入企劃的既有設定與內容應保留。
3. 完成網站基本資料、聯絡方式、Logo、圖片、分類、主題、首頁、內容頁、導覽與搜尋資料。
4. 使用 Chrome 檢查正式首頁及所有新增或更新頁面。
5. 分別檢查桌面版與手機版。
6. 修正破圖、錯誤連結、文字溢出、選單失效、對比不足及間距不一致。
7. 確認商品分類能持續增加子分類、父子關係正確，且商品只放在葉節點。
8. 確認網站只有一個作用中的 Logo，且未出現在一般素材庫。
9. 完成後統一回報已更新項目、頁面網址、使用素材及確實無法完成的外部限制。
